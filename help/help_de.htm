TemplateVar_XH 0.1 - Hilfe

Plugin entwickelt für CMSimple_XH 1.6.1* auf der Basis von Morepagedata_XH 1.2 von svasti

TemplateVar_XH ermöglicht es im Backend Variablen zu definieren, die im Template eingesetzt und ausgewertet werden können. So ist es z.B. möglich verschiedene Bestandteile des Templates, wie Header, Login-Link oder andere Elemente je nach Bedarf ein- oder auszuschalten. Auch der Wechsel von im Template verwendeten Bildern (z.B. Logo) ist möglich.

In der 'stylesheet.css' des Plugins sind Klassen definiert, damit die Ansicht von TemplateVar_XH im Backend an das jeweilige Template angepasst werden kann.

Voraussetzungen

TemplateVar_XH ist ein Plugin für CMSimple_XH. Es benötigt CMSimple_XH ≥ 1.6

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH Wiki finden sie ausführliche Hinweise.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Computer.
  3. Laden Sie das gesamte Verzeichnis templateVar/ auf Ihren Server in das plugins/ Verzeichnis von CMSimple_XH hoch.
  4. Vergeben Sie Schreibrechte für die Unterverzeichnisse css/, config/ und languages/.

Einstellungen

Die Konfiguration des Plugins erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Homepage. Wählen Sie Plugins → TemplateVar.

Unter 'Variablen' können Sie die Variablen neu anlegen, ändern oder löschen.

Unter 'Stylesheet' kann das Aussehen der von TemplateVar_XH angepasst werden.

Unter 'Konfiguration können Sie die Standardvariablen von TemplateVar_XH ändern.

Unter 'Sprache' kann die Lokalisierung vorgenommen werden. Sie können die Zeichenketten in Ihre eigene Sprache übersetzen (falls keine entsprechende Sprachdatei zur Verfügung steht), oder sie entsprechend Ihren Anforderungen anpassen.

Beim Überfahren der Hilfe-Icons mit der Maus werden Hinweise zu den jeweiligen Einstellungen angezeigt.

Verwendung der Variablen

Um TemplateVar_XH zu nutzen können im Template folgende Funktionen verwendet werden:
<?php if (templateVarCheck('Variablenname')): ?>
entsprechender HTML-Code
<?php endif ?>

Mit dieser Funktion wird je Status der entsprechenden Variable der Bestandteil des Templates angezeigt, oder eben nicht.

Mit der zweiten Funktion "templateVar('Variablenname')" wird der Wert der gewählten Variable ausgegeben
<?php echo (templateVar('logo')?>

Beispiel
<img alt="Logo" title="Logo" scr="<?php echo templateVar('logo')?>" >

Selbstverständlich können auch beide Funktionen kombiniert werden:
<?php if (templateVarCheck('show_logo')): ?>
<img alt="Logo" title="Logo" scr="<?php echo templateVar('logo')?>" >
<?php endif ?>

In diesem Beispiel kann im Backend das Logo geändert werden und auch die Anzeige manipuliert werden, ohne den Quelltext des Templates ändern zu müssen

Die dritte Funktion ermöglicht die Anzeige von Elementen im Template in einem bestimmten Zeitraum:

<?php if (templateVarDate()):
echo 'Dieser Text wird zwischen dem ' . strftime("%d.%m.%Y",strtotime($plugin_cf['templateVar']['date_start'])) . ' und dem ' . strftime("%d.%m.%Y",strtotime($plugin_cf['templateVar']['date_end'])) . ' angezeigt';
endif ?>

Verwaltung der Variablen

Erste Variable anlegen

Beim ersten Pluginaufruf ist die Variablenliste leer und Sie können diese unter dem Punkt 'Variablen' anlegen, falls Sie welche benötigen. In das erste Feld wird der Variablenname eingegeben, der in den Funktionen verwendet wird. Variablennamen müssen eindeutig sein.

Weitere Variablen können duch Klick auf die Schaltfläche "+" angelegt werden. Verschieben geht über das "Pfeilsymbol" und Löschen mit dem Löschsymbol.

Den Variableninhalt kann man ändern, wenn man über den Link "Variablenwerte" oder das Edit-Symbol das Eingabeformular aufruft.

Alle Änderungen (anlegen, ändern, verschieben und löschen) werden nur nach Bestätigung mit der Schaltfläche 'sichern' übernommen.

Verfügbare Feld-Typen

Variablenwerte ändern

Checkbox

Kann irgendetwas an- oder ausschalten, wie in obigem Beispiel die Breite des Inhaltsbereichs. Ein anderes Beipiel, um das Einlog-Feld von Memberpages zu zeigen oder nicht zu zeigen:

<?php if(isset($memberslogin)) {echo memberslogin();} ?>

Color picker

nutzt Jan Odvárko's jscolor. Color Picker erzeugt Farbcode der mit # anfängt. Statt Farbcode kann man auch englische Farbnamen (z.B.: white, red, blue, green, black) eingeben.

Image folder

zeigt eine Auswahlliste mit allen Bildern aus dem Ordner, der in der Pluginkonfiguration angegeben ist.

Input field

erzeugt einen selbstwachsenden Eingabebereich für Fälle, in denen die anderen Feldtypen zu begrenzt sind.

Option list

Bei Auswahl dieses Typs im Pluginbackend erscheint eine neue Zeile in der man die Optionswerte einträgt, entweder so
option 1||option 2||option 3 oder so
option 1|angezeigter Text 1|option 2|angezeigter Text 2|option 3|angezeigter Text 3 etc.
Kein | am Ende!

Template image

erzeugt eine Auswahlliste mit allen Bildern des Template Ordners. Den Pfad in der Pluginkonfiguration eingeben.
